### Implementation Notes
1. **Model invocation:** `stable-audio-open-1.0` via Hugging Face pipeline or Stability’s reference repo. Use 44.1kHz stereo WAV output, 40–45s duration, and temperature 1.0 for organic movement.
2. **Looping:** Post-process in DAW to create seamless loops (crossfade tail/head ~2s). Normalize to -16 LUFS for ambience.
3. **File naming:** Stick to snake_case, already assigned above. If multiple variations needed, append `_v2`, `_v3`, etc.
4. **Distribution:** Store in `public/audio/ambience/<category>/` to allow Vite static serving and caching.
5. **Metadata:** Add entries to any audio manifest JSON if required by runtime loader.

Feel free to expand categories (e.g., romance, mystery, sports) by cloning the format. Stability’s open model can be fine-tuned with our gameplay captures for even tighter stylistic control.
